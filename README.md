# DiffTime CLI

A tiny CLI tool inspired by the Unix philosophy that calculates the difference between two dates.

The order of the dates does not matter.

---

Format:
DD-MM HH:MM

Example:
difftime "07-03 10:15" "07-03 18:40"

Output:
8h 25m
